# Color Palette Documentation

This document describes the color palette used throughout the portfolio site.

## Primary Colors

### Primary (Indigo/Purple)
- **Primary 500**: `#6366f1` - Main brand color for links and accents
- **Primary 600**: `#4f46e5` - Hover states
- **Primary 700**: `#4338ca` - Active states

Used for: Primary links, buttons, and brand elements

## Accent Colors

These colors are used for "colorful texts" and visual variety:

- **Cyan**: `#06b6d4` - GitHub links, tech highlights
- **Purple**: `#a855f7` - Blog links, creative elements
- **Pink**: `#ec4899` - Personal content, highlights
- **Orange**: `#f97316` - Tags, warnings
- **Green**: `#10b981` - Success states, positive indicators
- **Blue**: `#3b82f6` - LinkedIn links, info elements

## Neutral Colors

- **Neutral 50**: `#fafafa` - Background
- **Neutral 100**: `#f5f5f5` - Subtle backgrounds
- **Neutral 200**: `#e5e5e5` - Borders
- **Neutral 500**: `#737373` - Secondary text
- **Neutral 600**: `#525252` - Muted text
- **Neutral 700**: `#404040` - Body text
- **Neutral 900**: `#171717` - Headings

## Semantic Colors

- **Success**: `#10b981` - Published posts, positive actions
- **Warning**: `#f59e0b` - Drafts, caution states
- **Error**: `#ef4444` - Error messages, destructive actions
- **Info**: `#3b82f6` - Informational content

## Usage Guidelines

1. **Links**: Use primary colors for main navigation, accent colors for variety
2. **Text**: Use neutral colors for hierarchy (900 for headings, 700 for body, 500 for secondary)
3. **Backgrounds**: Use neutral-50 for main background, white for cards
4. **Borders**: Use neutral-200 for subtle separation
5. **Tags**: Use accent colors with low opacity backgrounds for visual interest

## Tailwind Classes

- `text-primary-600` - Primary link color
- `text-accent-{color}` - Accent colors (cyan, purple, pink, etc.)
- `bg-neutral-50` - Main background
- `border-neutral-200` - Standard borders
- `text-gradient` - Gradient text utility class
